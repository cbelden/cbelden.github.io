Recent Projects

redditsearch.io

Users of the popular social news site, Reddit, probably know how cruddy the search feature is. Reddit Search aims to remedy this using modern ranking algorithms and post meta data to more accurately estimate each link's relevancy.

pybeacon

My senior design group created a Bluetooth beacon logging device, trackr, and I designed the device's software package, pybeacon. The trackr device listens for proximal Bluetooth beacons and records a timestamp and beacon ID.

markovchain

I created a custom Markov chain package to use in some of my other projects. The MarkovChain class constructs a Markov chain using a text corpus and then generates phrases by performing a random walk.

lyricfetch

I've discovered that programmatically retrieving full song lyrics using online APIs is quite convoluted. To work around this, I created lyricfetch to scrape full lyric listings from the Lyric Wikia website.

lyricscramble Twitter bot

The lyricscramble Twitter bot (@lyricscramble) listens to song requests, then replies with a scrambled phrase from the requested song. You can check it out right now by tweeting: "@lyricscramble artist -> title"
