Recent Projects

redditsearch.io

Users of the popular social news site, Reddit, probably know how cruddy the search feature is. Reddit Search aims to remedy this using modern ranking algorithms and post meta data to more accurately estimate each link's relevancy.

lyricscramble

Lyric Scramble constructs a Markov chain trained on user specified song lyrics and composes silly messages. Should have a RESTful API up and running soon!

pybeacon

My senior design group created a Bluetooth beacon logging device, trackr, and I designed the device's software package, pybeacon. The trackr device listens for proximal Bluetooth beacons and records a timestamp and beacon ID.
